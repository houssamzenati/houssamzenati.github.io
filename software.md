I develop software that supports my research in causal inference, sequential decision-making, and machine learning, ranging from reusable packages to research code released alongside specific papers.

## Open-source libraries:

<strong><a href="https://judithabk6.github.io/med_bench/">med_bench</a></strong> is a Python package for causal mediation analysis that implements the most common estimators within a unified framework, while allowing flexible choices for nuisance parameter models. It accompanies the paper *Causal mediation analysis with one or multiple mediators: a comparative study*.
